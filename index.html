HRA Design System (Work in Progress)

The Human Reference Atlas (HRA) design system presents logos, icons, style guides, and web components for use in creating applications using the HRA.

WARNING: this initial site is a work in progress. However, the icons and logos are of high quality and ready for use. Click an icon to get the CDN url to use. You can see the latest developments in via Figma here.
